# blog-informatorio

## Cómo iniciar el proyecto

### Comandos generales

Copiar `.env.example` a `.env`:
```sh
cp .env.example .env
```

Iniciar servidor de desarrollo (dentro de tu [`virtualenv`](#pip-y-virtualenv)):
```sh
python django-project/manage.py runserver
```
